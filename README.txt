SDG 11 Transport Optimization Project

To run this project:

1. Install requirements:
   pip install -r requirements.txt

2. Run Jupyter notebook:
   jupyter notebook transport_optimization.ipynb

3. OR Run Python script:
   python transport_optimization.py

Required libraries:
- pandas, numpy, matplotlib, seaborn
- scikit-learn, jupyter

The notebook will generate all analysis and visualizations automatically.
